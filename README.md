# ✈️ Passagens Bot - DevOps
Projeto prático de automação de infraestrutura.

## 🛠️ Tecnologias Usadas
- **Docker**: Containerização do Bot.
- **Kubernetes & Helm**: Orquestração e gerenciamento de pacotes.
- **GitHub Actions**: Automação de CI/CD.

## 🚀 Como testar
Para validar o Helm Chart localmente:
`helm lint helm/passagens-bot/`
